# Report Contract

Use this before Steps 7-8.

## Artifact Set

Write these files:

1. `bazi_data_audit.md`
2. `bazi_overview.md`
3. `bazi_classics_audit.md`
4. `bazi_timing_report.md`
5. `bazi_life_report.md`
6. `bazi_appendix.md`

## Voice

- Professional, plain Chinese.
- Explain the lived meaning first, then show evidence.
- Use “倾向”“更容易”“在某些条件下”“需要看大运是否引动”.
- Avoid “必定”“注定”“绝对”“一定会”.
- Keep technical labels visible but translated.

## Evidence Standard

Every major claim must cite at least one of:

- pillar or palace evidence from structured data
- month command / exposed stem evidence
-调候 layer evidence
-格局 layer evidence
-气势 layer evidence
- luck cycle evidence with age

## Life Stage Rules

- Child subject: write to parent/guardian. Focus on temperament, learning environment, health rhythm, family support, long-term potentials. Avoid present-tense marriage, wealth, and adult career claims.
- Teen subject: focus on development, study style, emotional regulation, parent-child communication, early direction. Avoid fixed career or marriage conclusions.
- Adult subject: career, wealth, relationship, health rhythm can be discussed as tendencies and timing windows.
- Unknown audience: stop and ask; do not assume self-reading.

## Section Minimums

`bazi_life_report.md` should include:

```markdown
# 八字人生报告

## 1. 核心性格与能力结构
## 2. 学习与思考方式
## 3. 事业方向与工作环境
## 4. 财务模式
## 5. 关系、伴侣与家庭互动
## 6. 健康倾向与生活节律
## 7. 当前阶段建议
```

Each section needs:

1. one plain-language conclusion
2. one evidence paragraph
3. one practical suggestion

## Final Audit

Before finalizing, check:

- The report never uses a single merged喜用神 without layer labels.
- Any age or year range matches `bazi_report_context.md`.
- High-stakes advice is framed as cultural reference.
- The appendix can reproduce the evidence trail.
