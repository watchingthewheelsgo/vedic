# 跨盘接触与相位判定规则 (aspect-policy)

> 合盘 Layer 2「方向性叠盘」的判据底座。
> 体系：纯 Parashari / KN Rao。只用吠陀判据，不引入西方占星。
> 数据来源：双方 structured_data.md 的 D1 行星位置 + 宫主表 + 尊贵度 + 吠陀视相关系表。

---

## 0. 方向铁规

所有跨盘接触必须写成方向句，A→B 与 B→A **分开各算一次，禁止合并**：

```
A 的 [行星]([P1角色/Karaka]) → B 的 [N宫/点位]
  A 发出什么 | B 如何体验
```

判据按权重分三层：**整宫落点 > Graha Drishti > 度数接近度**。
度数只调强弱，不独立成立、不否决。缺度数数据时退回整宫判据，不阻塞。

---

## 1. 整宫落点（主判据，权重最高）

A 的某行星落在 B 的某星座 = 落入 B 的对应宫位（从 B 的 Lagna 星座起算）。

- 含义：A 把这颗行星的能量带进 B 的该人生领域，激活该宫主题。
- 计算：`B宫位 = (A行星星座编号 − B的Lagna星座编号) mod 12 + 1`。
- 解读用 B 的宫位领域 + A 行星的 P1 角色/Karaka + A 行星在自盘的尊贵度。
- 关键落宫（提案优先级）：落入 B 的 1 / 4 / 5 / 7 / 8 / 11 / 12 宫 = 强信号。

> 示例：A 的 Saturn（B盘视角是外来星）落入 B 的 7宫
> → A 给 B 的婚姻领域带来"责任/约束/沉淀"，B 体验为"被要求成熟、节奏变慢"。
> A 的 Saturn 在自盘是否入庙/落陷，决定这份约束是"稳固"还是"压制"。

---

## 2. Graha Drishti（特殊视相，第二判据）

吠陀整宫视相，**不是**西方度数相位。A 的行星视相落到 B 的宫/点位 = A 对 B 该领域形成注视。

| 行星 | 视相宫位（从所在宫起算） |
|------|------------------------|
| 全部行星 | 7th（对宫） |
| Mars | 额外 4th、8th |
| Jupiter | 额外 5th、9th |
| Saturn | 额外 3rd、10th |
| Rahu/Ketu | 保守取 7th（对宫）；5/9 式视相为流派分歧，**暂不启用，待案例校准** |

- 凶星（Saturn/Mars/Rahu）视相 → 施压 / 限制 / 激化。
- 吉星（Jupiter/Venus/Moon）视相 → 保护 / 扩张 / 滋养。
- 吉凶判定叠加该星在自盘的复合尊贵度（落陷凶星视相 = 破坏性更纯）。

> Rashi Drishti（星座相视，Jaimini）：仅在 DK/UL 等 Jaimini 点位交互时作补充，
> 不与 Graha Drishti 混用于同一条判定。

---

## 3. 度数接近度（强弱标注，非独立判据）

吠陀不设西方 orb 档位。度数只给「同宫合相 / 对宫」标强弱：

| 条件 | 标注 |
|------|------|
| 同宫且度数差 < 5°（发光体参与放宽到 < 7°） | [紧密] 能量交融最强 |
| 同宫且度数差 5°–12° | [一般] |
| 同宫 > 12°，或仅整宫同座 | [整宫接触] |

对宫接触以 180° 为基准算差，分级同上。

⚠️ 度数只调强弱，**不单独成立一个接触，也不否决**。无度数数据 → 退回整宫判据。

## 4. 接触性质分类（喂给 interpretation-rubric 的六维矩阵）

每条接触落到一个性质桶，禁止把单一凶星接触直接写成"失败"：

- 舒适吸引：Moon-Venus、Moon-Jupiter、Venus-Jupiter 的柔性接触
- 性/行动吸引：Venus-Mars、Mars 触 5/7/8 宫
- 放大/理想化：Rahu 触 Moon/Venus/Lagna/7宫轴
- 黏着/失衡/强迫：Rahu 紧密 + 凶星叠加、8/12 宫落点
- 约束/承载：Saturn 触 Moon/Venus/Lagna/7宫主
- 摩擦/修复：Mars-Mercury、Mars-Moon、6/8/12 宫落点
  → 每个风险接触必须同时检查修复资源（吉星视相 / 宫主强 / Dasha 支持）

---

## 5. 禁止事项（防止混入"奇怪的东西"）

- 不引入西方 orb 体系（多档容许度、入相位/出相位、容许度递减表）
- 不引入西方相位角（六分 60° / 梅花 150° / 西方度数 trine 120°）
  —— 吠陀的「三合/三角」是整宫 1/5/9 同元素，不是 120° 角
- 不引入 composite 中点合成盘 / Davison 盘
- 不用"金星合月=灵魂伴侣"这类西方 synastry 话术，一律翻成宫主/Karaka/尊贵度语言
- 跨盘 DK 一律 7K 主表，不引入 8K/Rahu DK
- 所有接触的解读起点必须是 P1 角色或尊贵度，度数/SAV 只作辅助确认
