# 友谊框架 (friendship-framework)

> 适用：朋友、知己、同伴、搭子。
> 进入：Layer 0.5 平扫后 intake 选「友谊」。
> **性别**：无性别角色，纯 A/B 方向。

---

## Layer 0 双盘资格：读什么

- **D1**：3宫(同伴/胆识)、11宫(友群/社交/共同愿望)、Moon(情绪共鸣)、Mercury(交流)、Lagna(气质合拍)
- **Dasha**：当前是否激活 3/11宫
- 承载卡四问：能提供的陪伴/共鸣 / 需要什么 / 边界在哪 / 当前是否在愿意投入社交的阶段

## Layer 2 重点叠盘信号

- 情绪共鸣：Moon-Moon、Moon-Mercury
- 心智默契：Mercury-Mercury、Jupiter(价值观/三观)
- 社交/共同愿望：3/11宫互落
- 善意托底：Jupiter 互照
- 摩擦：Mars-Mercury、6/8/12宫落点
- **不侧重 Venus-Mars 性张力**（友谊不以此论）

## 六维矩阵侧重

高权重：**情绪安全 / 沟通修复 / 现实协作（共同活动）**。
- "吸引与亲密"改为"**亲和/默契**"（非性向）。
- "长期承载"看"友谊存续 / 边界健康度"。

## 本框架关系类型词

知己型 / 互相支持型 / 泛泛之交 / 兴趣搭子 / 易消耗型

---

## 关系阶段/形态适配（可选；未指定 → 通用读法）

- **刚认识**：看初始亲和（Moon/Lagna 合拍度）。
- **一般朋友 / 深交知己**：看 Moon-Moon / Mercury 共鸣的深度。
- **渐远**：看是否缺持续激活（3/11宫弱）或有摩擦累积。
- **单方想靠近**：方向不对称，诚实说是单向，不替对方表态。
