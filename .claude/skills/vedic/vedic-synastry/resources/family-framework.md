# 家人框架 (family-framework)

> 适用：父母子女、家人、亲属合盘。
> 进入：Layer 0.5 平扫后 intake 选「家人/亲子」。
> **方向**：用代际方向（长辈→晚辈 / 晚辈→长辈）或 A/B；按实际家庭角色（父/母/子/女）解读。非二元同样按 A/B 中性，不强套性别。

---

## Layer 0 双盘资格：读什么

- **D1**：4宫(母/家庭/情感根基)、9宫(父/恩泽)、Moon(母性/情绪)、Sun(父性/权威)、5宫(子女)、Lagna
- **子女 Karaka**：Jupiter
- **Dasha**：当前是否激活 4/9/5宫
- 承载卡四问：能提供的照护/支持 / 需要什么 / 代际压力下如何（控制？疏离？）/ 当前家庭关系处于什么阶段

## Layer 2 重点叠盘信号

- 亲密照护：Moon 跨盘(情感联结)、4宫互落
- 权威/恩泽：Sun 跨盘、9宫互落
- 责任/代沟：Saturn 触对方 Moon/Sun/4宫（代际压力、距离感）
- 庇佑：Jupiter 触对方 Lagna/5宫
- 纠缠/业力：Rahu-Ketu 触 Moon/Sun/4宫
- 摩擦：Mars 触 4宫、6/8/12宫落点

## 六维矩阵侧重

高权重：**情绪安全（家庭温度）/ 长期承载（血缘恒久）**。
- "吸引与亲密"改为"**亲密照护/情感联结**"（非性向）。
- "现实协作"看家庭事务分担。
- "当前时机"看代际关系当前的张力/缓和期。

## 本框架关系类型词

滋养型 / 责任型 / 代沟疏离型 / 纠缠依附型 / 互相支撑型

---

## 关系阶段/状态适配（可选；家人是既定关系，看当前状态而非"建不建立"）

- **亲近**：Moon/4宫和谐为主。
- **疏离**：看 Saturn/Ketu 距离感、缺情感激活。
- **冲突**：Mars / 6/8宫摩擦 + 代际 Saturn 压力。
- **和解中**：看修复资源（Jupiter 庇佑、宫主强）。
