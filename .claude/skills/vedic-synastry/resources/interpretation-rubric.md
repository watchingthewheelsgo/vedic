# 六维矩阵判定词典 (interpretation-rubric)

> 合盘最终结论规则。六维矩阵，**不给单一匹配度总分**，不用好坏二分。
> 体系：KN Rao / Parashari。证据语言一律宫主(P1角色) + Karaka + 复合尊贵度 + SAV(极端值) + Dasha。
> 输入：Layer 0(资格) + Layer 1(月宿) + Layer 2(方向叠盘) + Layer 3(时机) 的结论。

---

## 0. 证据权重（高→低）

D1 方向性叠盘 > UL·DK 交互 > D9 承载 > Dasha 时机 > 月宿 Ashtakoota（仅筛查）。
**无单项一票否决**；但 Layer 0 判定一方承载不足 → 整体强制打"条件式"标签。

## 2. 每维度必含五要素

1. 支持证据（方向句 + 判据）
2. 制约证据（同样列出，禁止只挑一边）
3. 方向差异（A→B vs B→A 不对称处）
4. 置信度（高/中/低，受双方时间精度与降级影响）
5. 现实验证点（一个用户可观察的现实现象，用于事后校验，不反向生成结论）

---

## 3. 最终关系类型（不二分，按矩阵组合选）

- 高吸引 + 高承载
- 高吸引 + 低承载
- 低戏剧性 + 高承载
- 成长/催化型
- 现实协作型
- 节奏错位型
- 高消耗且修复资源不足

每个风险维度必须同时列修复资源（吉星视相 / 宫主强 / Dasha 支持），
禁止把单一凶星接触直接写成"失败"。

---

## 4. 红线（人工判断，不由占星替代）

暴力、控制、财务欺骗、权力严重不对等属现实安全议题，
盘面只提示风险结构，最终判断交给现实，不用占星结论替代。
